Este es el listado de comandos que he aprendido y estoy tratando de dominar

  1. git log --oneline

    2. Te muestra el id commit y el titulo del commit.

  2. git log --decorate

    3. Te muestra donde se encuentra el head point en el log.

  3. git log --stat

    4. Explica el numero de lineas que se cambiaron brevemente.

  4. git log -p

    5. Explica el numero de lineas que se cambiaron y te muestra que se cambio en el contenido.

  5. git shortlog

    6. Indica que commits ha realizado un usuario, mostrando el usuario y el titulo de sus commits.

  6. git log --pretty=format:"%cn hizo un commit %h el dia %cd"

    7. Muestra mensajes personalizados de los commits.
